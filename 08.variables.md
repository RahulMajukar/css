# 🎛️ CSS Variables (Custom Properties) - Dynamic Styling

**CSS Variables** = Reusable values you can change anywhere!

**Think of them as:** "Find & Replace" for your entire stylesheet  
**Old way:** Hardcode colors everywhere ❌  
**New way:** Change once, updates everywhere ✅

## Step 1: Declare Variables

**`:root`** = Global scope (best practice):

```css
:root {
  /* Colors */
  --primary-color: #007bff;
  --secondary-color: #6c757d;
  --success-color: #28a745;
  
  /* Spacing */
  --spacing-xs: 0.5rem;
  --spacing-sm: 1rem;
  --spacing-md: 2rem;
  --spacing-lg: 4rem;
  
  /* Typography */
  --font-main: "Inter", system-ui, sans-serif;
  --font-size-base: 1rem;
  
  /* Border radius */
  --border-radius: 8px;
}
```

## Step 2: Use Variables

**Two syntaxes:**

```css
.button {
  background: var(--primary-color);
  padding: var(--spacing-md) var(--spacing-lg);
  font-family: var(--font-main);
  border-radius: var(--border-radius);
  color: white;
}
```

**Default fallback:**
```css
color: var(--text-color, #333);  /* Use #333 if --text-color missing */
```

## 🎨 Dynamic Theming (Dark/Light Mode)

**1. Light theme (default):**
```css
:root {
  --bg-color: #ffffff;
  --text-color: #333333;
  --card-bg: #f8f9fa;
}
```

**2. Dark theme:**
```css
[data-theme="dark"] {
  --bg-color: #1a1a1a;
  --text-color: #ffffff;
  --card-bg: #2d2d2d;
}
```

**3. Apply everywhere:**
```css
body {
  background: var(--bg-color);
  color: var(--text-color);
}

.card {
  background: var(--card-bg);
}
```

**4. Toggle with JavaScript:**
```javascript
// Switch to dark mode
document.documentElement.setAttribute('data-theme', 'dark');
```

**HTML:**
```html
<body data-theme="dark">
  <!-- Variables update automatically! -->
</body>
```

## 🧮 Math with CSS Variables

**`calc()` + Variables = Responsive magic:**

```css
:root {
  --sidebar-width: 280px;
  --spacing-lg: 2rem;
}

.sidebar {
  width: var(--sidebar-width);
}

.main {
  /* Full width minus sidebar minus spacing */
  width: calc(100vw - var(--sidebar-width) - var(--spacing-lg));
}

.card {
  /* Responsive padding */
  padding: calc(var(--spacing-sm) * 2);
}
```

**Modern math functions:**
```css
--container-width: min(100%, 1200px);
--responsive-font: clamp(1rem, 4vw, 2rem);
```

## 🔄 Dynamic Values (JavaScript Control)

**Update variables at runtime:**

```css
:root {
  --progress: 0%;
}

.progress-bar {
  width: var(--progress);
  transition: width 0.3s ease;
}
```

```javascript
// Animate progress
function updateProgress(percent) {
  document.documentElement.style.setProperty('--progress', percent + '%');
}
```

## 🌙 Advanced: CSS `prefers-color-scheme`

**Automatic theme detection:**

```css
/* Light theme (default) */
:root {
  --bg: white;
  --text: #333;
}

/* Dark theme */
@media (prefers-color-scheme: dark) {
  :root {
    --bg: #1a1a1a;
    --text: white;
  }
}

/* User override */
[data-theme="dark"] {
  --bg: #121212 !important;
  --text: #e0e0e0 !important;
}
```

## 🚀 Complete Theme Switcher Example

**CSS:**
```css
:root {
  --bg-primary: #ffffff;
  --bg-secondary: #f8f9fa;
  --text-primary: #212529;
  --border-color: #dee2e6;
}

[data-theme="dark"] {
  --bg-primary: #121212;
  --bg-secondary: #1e1e1e;
  --text-primary: #e0e0e0;
  --border-color: #373737;
}

body {
  background: var(--bg-primary);
  color: var(--text-primary);
  transition: background 0.3s, color 0.3s;
}

.card {
  background: var(--bg-secondary);
  border: 1px solid var(--border-color);
}
```

**HTML + JS:**
```html
<!DOCTYPE html>
<html data-theme="light">
<body>
  <button onclick="toggleTheme()">Toggle Theme</button>
  <div class="card">This card changes color!</div>
  
  <script>
    function toggleTheme() {
      const html = document.documentElement;
      const current = html.getAttribute('data-theme');
      html.setAttribute('data-theme', current === 'dark' ? 'light' : 'dark');
    }
  </script>
</body>
</html>
```

## ✅ CSS Variables Checklist

```
✅ Declared in :root (global)
✅ Used with var(--name)
✅ Fallback values: var(--name, fallback)
✅ Theme switching with [data-theme]
✅ Responsive with calc() + clamp()
✅ Spacing system (0, 4px, 8px, 16px...)
✅ Color scale (50, 100, 500, 900)
```

## ❌ Common Mistakes

1. **Missing colon:** `--color: red` ❌ → `--color: red;` ✅
2. **No fallback:** `color: var(--missing-color)` → pink!
3. **Scoped wrong:** Don't declare inside `.class` (use `:root`)
4. **Forgetting transition** for theme changes
