# 🌐 CSS Grid - 2D Layout Superpowers

**Flexbox** = 1D layouts (row OR column)  
**Grid** = 2D layouts (rows AND columns)  

**Perfect for:** Cards, dashboards, photo galleries, forms, entire pages!

## Step 1: Turn ON Grid

```css
.container {
  display: grid;  /* ⭐ That's it! */
}
```

---

## 🎯 Core Grid Properties (Master These 5)

### 1. `grid-template-columns` & `grid-template-rows`

**Define your grid tracks:**

```css
.grid {
  display: grid;
  
  /* 3 columns: 200px + 1fr + auto */
  grid-template-columns: 200px 1fr auto;
  
  /* 2 rows: both flexible */
  grid-template-rows: 1fr 1fr;
}
```

**Column/Row Units:**
```
200px     → Fixed 200px
1fr       → 1 fraction of available space
2fr       → 2 fractions (twice as wide)
auto      → Content size
minmax(200px, 1fr) → Min 200px, max flexible
repeat(3, 1fr)     → 3 equal columns
```

**Examples:**
```css
/* 3 equal columns */
grid-template-columns: repeat(3, 1fr);

/* 4 columns: mobile/desktop friendly */
grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));

/* Sidebar layout */
grid-template-columns: 250px 1fr;
```

### 2. `gap` - Perfect Spacing

```css
.gallery {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 2rem 1rem;  /* row-gap column-gap */
}
```

**Old way vs New way:**
```css
/* ❌ Old */
grid-row-gap: 2rem;
grid-column-gap: 1rem;

/* ✅ New */
gap: 2rem 1rem;
```

### 3. `justify-items` & `align-items`

**Content alignment INSIDE grid cells:**

| Property | Horizontal | Vertical |
|----------|------------|----------|
| `justify-items` | Left/Center/Right | - |
| `align-items` | - | Top/Center/Bottom |

```css
.cards {
  display: grid;
  justify-items: center;   /* Content centered horizontally */
  align-items: center;     /* Content centered vertically */
}
```

### 4. `justify-content` & `align-content`

**Entire grid alignment inside container:**

```css
.dashboard {
  display: grid;
  justify-content: center;  /* Grid centered horizontally */
  align-content: start;     /* Grid packed to top */
}
```

## 🎮 Grid Item Placement (Superpowers!)

### `grid-column` & `grid-row`

**Place items by coordinates:**

```css
.header {
  grid-column: 1 / -1;  /* Span all columns */
  grid-row: 1;
}

.sidebar {
  grid-column: 1;
  grid-row: 2 / 4;  /* Span rows 2-3 */
}

.main {
  grid-column: 2;
  grid-row: 2 / 4;
}
```

**Shorthand:**
```css
/* grid-column-start / grid-column-end */
.header { grid-column: 1 / -1; }  /* Full width */
.sidebar { grid-area: 2 / 1 / 4 / 2; }  /* row-start / col-start / row-end / col-end */
```

### `grid-area` - Named Magic

**1. Define areas:**
```css
.dashboard {
  display: grid;
  grid-template-areas: 
    "header header"
    "sidebar main" 
    "sidebar main";
}
```

**2. Assign names:**
```css
.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main { grid-area: main; }
```

---

## 🔥 Auto-Fit & Auto-Fill (Responsive Magic)

**Cards that automatically adjust:**

```css
.gallery {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  gap: 2rem;
}
```

**How it works:**
```
1 item → Full width
2 items → 2 columns  
3 items → 3 columns
4+ items → Responsive wrapping
```

**auto-fit vs auto-fill:**
```
auto-fit: Empty tracks shrink to zero
auto-fill: Empty tracks stay visible
```

## 🏗️ Complete Real-World Examples

### 1. **Product Gallery** ⭐ Most Popular
```css
.products {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(280px, 1fr));
  gap: 2rem;
  padding: 2rem;
}

.product-card {
  padding: 2rem;
  border: 1px solid #eee;
  border-radius: 12px;
}
```

### 2. **Dashboard Layout**
```css
.dashboard {
  display: grid;
  grid-template-columns: 250px 1fr 300px;
  grid-template-rows: 80px 1fr 100px;
  grid-template-areas: 
    "header header header"
    "sidebar main widgets"
    "footer footer footer";
  min-height: 100vh;
  gap: 1rem;
}

.header { grid-area: header; background: #333; }
.sidebar { grid-area: sidebar; background: #f5f5f5; }
.main { grid-area: main; }
.widgets { grid-area: widgets; }
.footer { grid-area: footer; }
```

### 3. **Photo Gallery (Masonry Style)**
```css
.gallery {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
  grid-auto-rows: 20px;  /* Tiny rows */
  gap: 1rem;
}

.gallery img {
  grid-row: span 15;  /* Span ~300px height */
  width: 100%;
  height: 100%;
  object-fit: cover;
}
```

### 4. **Form Layout**
```css
.form-grid {
  display: grid;
  grid-template-columns: 150px 1fr;
  gap: 1rem 2rem;
  align-items: end;
}

.form-grid label {
  font-weight: 500;
}

.form-grid input,
.form-grid select {
  padding: 0.75rem;
  border: 1px solid #ddd;
  border-radius: 6px;
}
```

## 🔮 Subgrid (Modern Superpower)

**Child grids inherit parent tracks:**

```css
.parent {
  display: grid;
  grid-template-columns: 1fr 2fr 1fr;
}

.child {
  display: grid;
  grid-template-columns: subgrid;  /* ⭐ Inherits parent's columns */
}
```

**Use case:** Nested components that perfectly align.

---

## 📱 Mobile-First Grid

```css
.card-grid {
  display: grid;
  grid-template-columns: 1fr;  /* Mobile: 1 column */
  gap: 1.5rem;
}

/* Tablet */
@media (min-width: 768px) {
  grid-template-columns: repeat(2, 1fr);
}

/* Desktop */
@media (min-width: 1024px) {
  grid-template-columns: repeat(3, 1fr);
}
```

**OR Single line responsive:**
```css
.card-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
}
```

## 🎯 Grid vs Flexbox - When to Use What?

| Layout | Use Grid | Use Flexbox |
|--------|----------|-------------|
| Cards gallery | ✅ | ❌ |
| Navigation | ❌ | ✅ |
| Dashboard | ✅ | ❌ |
| Button group | ❌ | ✅ |
| Forms (label+input) | ✅ | ❌ |

**Rule:** 2D = Grid, 1D = Flexbox
